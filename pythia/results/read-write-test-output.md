TECHOPS-2187 aims to use AI bots (Claude, Gemini, GPT-4o) orchestrated via FlowSpec workflows to systematically inventory, prioritize, and generate XCTest unit tests for ~110K lines of largely untested legacy Objective-C code in the MyFitnessPal iOS app, with SonarQube enforcing 80% coverage on new code. The three-phase approach—inventory zero-coverage classes, estimate effort/testability, then scope-reduce by git activity and business criticality—needs careful quality gates to ensure generated tests validate meaningful behavior rather than just inflating line coverage, plus human checkpoints before merging and a strategy for flagging genuinely untestable classes that need refactoring first.
